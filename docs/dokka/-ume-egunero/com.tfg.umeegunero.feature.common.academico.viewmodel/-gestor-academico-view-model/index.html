GestorAcademicoViewModel

class GestorAcademicoViewModel @Inject constructor(centroRepository: CentroRepository, cursoRepository: CursoRepository, claseRepository: ClaseRepository) : ViewModel

ViewModel para la gestión académica de centros, cursos y clases.

Proporciona funcionalidad para cargar y gestionar la estructura académica de la aplicación, permitiendo navegar entre centros, cursos y clases, y gestionar las relaciones entre estos elementos.
