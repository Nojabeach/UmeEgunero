@Singleton

Repositorio para gestionar operaciones relacionadas con los cursos en la aplicación UmeEgunero.

Esta interfaz define los métodos para interactuar con los cursos, permitiendo operaciones como obtención, creación, actualización y eliminación de cursos.

Los cursos representan períodos académicos en los que se organizan las actividades educativas. Cada curso está asociado a un centro educativo y puede contener múltiples clases.
