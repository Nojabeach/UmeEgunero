Repositorio unificado para gestionar todas las comunicaciones en la aplicación

Este repositorio centraliza la lógica de todos los tipos de mensajes: notificaciones, chats, comunicados, incidencias, etc. en una interfaz común para simplificar la interacción con los diferentes sistemas de comunicación.
