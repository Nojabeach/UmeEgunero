RegistroDiarioRepository

@Singleton
class RegistroDiarioRepository @Inject constructor(firestore: FirebaseFirestore, localRegistroRepository: LocalRegistroActividadRepository, context: Context)

Repositorio para gestionar los registros diarios de actividades de los alumnos.

Esta clase maneja las operaciones CRUD para los registros de actividades, así como consultas específicas por alumno, clase, fecha, etc. Soporta almacenamiento offline mediante el repositorio local.

Estudiante 2º DAM

firestore

Repositorio local para acceso a la base de datos local

context

Contexto de la aplicación para verificar conectividad
