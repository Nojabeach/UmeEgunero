fun FormProgressIndicator(currentStep: Int, totalSteps: Int, modifier: Modifier = Modifier, showLabels: Boolean = true, showStepText: Boolean = true, progressColor: Color = MaterialTheme.colorScheme.primary, trackColor: Color = MaterialTheme.colorScheme.surfaceVariant, stepLabel: String = "Paso", height: Dp = 8.dp)

Componente que muestra un indicador de progreso para formularios multi-paso.

Este componente proporciona una barra de progreso visual que indica el avance del usuario a través de un formulario de varios pasos o un proceso secuencial. Puede mostrar opcionalmente etiquetas para el paso actual y el total de pasos, y es personalizable en términos de colores y tamaños.

Parameters

currentStep

El paso actual en el que se encuentra el usuario (comienza desde 1)

totalSteps

El número total de pasos en el formulario

modifier

Modificador opcional para personalizar el diseño del componente

showLabels

Si se deben mostrar etiquetas indicando el paso actual y el total (por defecto: true)

showStepText

Si se debe mostrar el texto "Paso X de Y" (por defecto: true)

progressColor

Color personalizado para la barra de progreso (por defecto: usa el color primario del tema)

trackColor

Color personalizado para el fondo de la barra de progreso (por defecto: usa el color de superficie del tema)

stepLabel

Etiqueta personalizada para mostrar antes del número de paso (por defecto: "Paso")

height

Altura personalizada para la barra de progreso (por defecto: 8.dp)
