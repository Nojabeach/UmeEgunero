EstadisticaCard

@Composable
fun EstadisticaCard(titulo: String, valor: String, descripcion: String, icono: ImageVector, iconoTint: Color = MaterialTheme.colorScheme.primary, modifier: Modifier = Modifier, valorColor: Color = MaterialTheme.colorScheme.primary, valorSufijo: String = "", onClick: () -> Unit? = null)

Componente que muestra una tarjeta de estadística con un valor destacado y su descripción.

Este componente se utiliza para presentar métricas importantes en los informes de rendimiento y uso de la aplicación. Cada tarjeta contiene un icono relacionado con la estadística, un valor numérico destacado y una descripción que contextualiza el dato mostrado.

Parameters

titulo

Título breve que describe la estadística

valor

Valor principal a mostrar (puede ser un número o porcentaje)

descripcion

Texto explicativo sobre el significado de esta estadística

icono

Icono que representa visualmente la categoría de esta estadística

iconoTint

Color personalizado para el icono (por defecto: color primario del tema)

modifier

Modificador opcional para personalizar el diseño del componente

valorColor

Color personalizado para el valor destacado (por defecto: color primario)

valorSufijo

Sufijo opcional a mostrar junto al valor (ej. "%", "pts")

onClick

Función opcional que se ejecuta al hacer clic en la tarjeta
