Pantalla de detalle de centro educativo.

Esta pantalla proporciona una interfaz completa para la visualización y gestión de los detalles de un centro educativo específico.

Características

  • Información detallada del centro

  • Acciones de gestión (editar, eliminar)

  • Integración con mapas y comunicaciones

  • Estados de carga y error

Funcionalidades

  • Visualización de datos del centro

  • Edición de información

  • Eliminación del centro

  • Acciones rápidas (llamada, email, ubicación)

Parameters

centroId

Identificador del centro a mostrar

viewModel

ViewModel que gestiona la lógica de administración

onNavigateBack

Callback para navegar hacia atrás

onNavigateToEdit

Callback para navegar a edición

onDeleteSuccess

Callback ejecutado tras eliminación exitosa
