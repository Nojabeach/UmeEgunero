AppModule

@Module
object AppModule

Módulo principal de inyección de dependencias para la aplicación UmeEgunero.

Este módulo se encarga de proporcionar las dependencias principales que no están directamente relacionadas con Firebase, como el repositorio de preferencias y el objeto SharedPreferences para el almacenamiento local.

Funcionalidades:

  • Proporciona el repositorio de preferencias para gestionar las configuraciones de usuario

  • Proporciona el objeto SharedPreferences para almacenamiento ligero de datos
