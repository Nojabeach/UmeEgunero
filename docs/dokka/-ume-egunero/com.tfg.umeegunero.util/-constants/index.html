Constants

object Constants

Constantes globales utilizadas en toda la aplicación.

Esta clase define valores constantes centralizados para evitar la duplicación y facilitar el mantenimiento. Incluye configuraciones para servicios de correo electrónico, endpoints de API, y constantes de presentación.

Nota: Por seguridad, los valores sensibles como tokens y claves se almacenan en el AndroidManifest.xml como meta-data o se generan dinámicamente a través de SecureTokenManager.
