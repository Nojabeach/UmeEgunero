class Converters

Clase de utilidad que proporciona convertidores para tipos de datos complejos.

Estos convertidores pueden utilizarse con Room o para cualquier otra conversión en la aplicación que requiera transformar entre tipos de datos nativos y formatos serializables o entre distintos tipos de datos.
