Servicio para enviar notificaciones por email usando un Google Apps Script.

Esta clase se encarga de la comunicación con un endpoint de Google Apps Script que procesa y envía emails utilizando plantillas predefinidas. Utiliza KTor Client para realizar las peticiones HTTP al servicio externo. Implementa seguridad robusta con tokens firmados y encriptados.
