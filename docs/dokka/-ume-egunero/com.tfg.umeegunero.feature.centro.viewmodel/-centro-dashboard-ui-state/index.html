Estado UI para la pantalla de dashboard del centro educativo.

Esta clase representa el estado completo de la interfaz de usuario para la gestión a nivel de centro educativo, conteniendo toda la información necesaria para renderizar correctamente la pantalla del dashboard.

Sigue el patrón de UI State en Jetpack Compose, donde el estado se mantiene de forma inmutable y cada cambio genera un nuevo objeto de estado.
