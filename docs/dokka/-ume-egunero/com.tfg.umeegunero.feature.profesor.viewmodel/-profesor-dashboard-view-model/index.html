ProfesorDashboardViewModel

class ProfesorDashboardViewModel @Inject constructor(usuarioRepository: UsuarioRepository, authRepository: AuthRepository, calendarioRepository: CalendarioRepository, registroDiarioRepository: RegistroDiarioRepository) : ViewModel

ViewModel para la pantalla de dashboard del profesor

Este ViewModel implementa el patrón MVVM (Model-View-ViewModel) y es responsable de toda la lógica de negocio relacionada con el dashboard del profesor:

  • Gestión del estado de la UI mediante StateFlow

  • Carga de datos desde los repositorios

  • Procesamiento de eventos del usuario

  • Manejo de operaciones asíncronas mediante corrutinas

El ViewModel actúa como intermediario entre la capa de datos (repositorios) y la capa de presentación (Composables), manteniendo la separación de responsabilidades y facilitando las pruebas unitarias.
