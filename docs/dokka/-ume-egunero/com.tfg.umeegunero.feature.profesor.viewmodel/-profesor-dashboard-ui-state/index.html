Estado UI para la pantalla de dashboard del profesor

Esta clase data contiene todas las propiedades necesarias para representar el estado completo de la interfaz de usuario del dashboard del profesor. Siguiendo el patrón de State en Jetpack Compose, esta clase inmutable permite una gestión eficiente y predecible del estado de la UI.
