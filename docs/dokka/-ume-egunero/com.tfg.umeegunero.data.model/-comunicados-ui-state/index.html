Estado de la UI para la pantalla de gestión de comunicados.

Esta clase encapsula todos los datos necesarios para la representación del estado de la interfaz de usuario en la pantalla de gestión de comunicados. Contiene tanto la lista de comunicados existentes como los estados del formulario para crear nuevos comunicados y los estados de envío.
