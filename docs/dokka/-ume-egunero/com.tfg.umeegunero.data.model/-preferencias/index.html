Preferencias

data class Preferencias(val idiomaApp: String = "es", val notificaciones: Notificaciones = Notificaciones(), val tema: TemaPref = TemaPref.SYSTEM)

Modelo que representa las preferencias configurables de un usuario en el sistema UmeEgunero.

Esta clase centraliza todas las opciones personalizables que afectan a la experiencia de usuario dentro de la aplicación. Las preferencias permiten adaptar el comportamiento e interfaz a las necesidades específicas de cada usuario, mejorando así la usabilidad y satisfacción.

El modelo incluye configuraciones relativas al idioma, las notificaciones y el tema visual de la aplicación, con valores predeterminados que se aplican para nuevos usuarios.
