Representa un usuario en la aplicación UmeEgunero.

Esta clase define la estructura de datos para los diferentes tipos de usuarios que pueden interactuar con la aplicación, como administradores, profesores, familiares y alumnos.

Cada usuario tiene un conjunto de propiedades que definen su perfil, rol y permisos dentro del sistema educativo.
