Familiar

data class Familiar(val id: String = "", val nombre: String = "", val apellidos: String = "", val parentesco: String = "")

Modelo que representa la información básica de un familiar vinculado a un alumno.

Esta clase contiene información resumida de un familiar para uso en listados o referencias rápidas. Para información completa del familiar, se debe consultar el modelo Usuario correspondiente usando el ID.
