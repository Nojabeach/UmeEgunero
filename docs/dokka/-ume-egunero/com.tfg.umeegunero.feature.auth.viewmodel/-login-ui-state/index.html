Estado UI para la pantalla de login.

Esta clase encapsula todos los estados posibles de la interfaz de usuario para la pantalla de inicio de sesión, siguiendo el patrón de State Hoisting recomendado para Jetpack Compose.

El state hoisting es un patrón de diseño en el que el estado se eleva (o "hoists") a un nivel superior en la jerarquía de componentes, de forma que podamos separar la lógica de negocio de la interfaz de usuario.
